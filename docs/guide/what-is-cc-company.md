# cc-company とは

cc-company は、Claude Code 上で**仮想組織**を構築・運営するプラグインです。

## コンセプト

```
あなた → 秘書（窓口） → 各部署（必要に応じて追加）
```

- **秘書**が常にあなたの窓口です
- TODO管理、壁打ち、メモ、何でも秘書に相談できます
- 仕事が増えてきたら、秘書が部署の追加を提案します

## なぜ秘書が中心なのか

従来の組織管理ツールは、最初に組織構造を決めてから運用を始めます。しかし、個人やスモールチームにとって、最初から全部署を揃える必要はありません。

cc-company は **「まず秘書だけ」** で始めます。

- タスクが増えたら → 秘書がTODO管理
- アイデアが浮かんだら → 秘書がメモを取る
- 壁打ちしたいとき → 秘書が相談相手に
- リサーチが必要になったら → 秘書が「リサーチ部門を作りましょうか？」と提案

**使いながら自然に組織が育っていく。** それが cc-company の設計思想です。

## 誰のためのツール？

- 個人開発者
- フリーランス
- 副業・スタートアップ
- 日々のタスクや情報を整理したい人

部署を増やせば、プロジェクト管理、マーケティング、経理、営業など、本格的な組織運営にも対応できます。
