# 秘書との日常

セットアップ後は、`/company` で秘書に話しかけるだけです。

## TODO管理

```
> /company
秘書: おはようございます！何かありますか？

> 今日やること教えて
秘書: 今日のTODOです:
  - [ ] クライアントAへ見積もり送付
  - [ ] LP設計書のレビュー

> 「Stripe決済の実装」を追加して、優先度高で
秘書: 追加しました！
  - [ ] Stripe決済の実装 | 優先度: 高
```

TODOは `secretary/todos/YYYY-MM-DD.md` に保存されます。

## 壁打ち・相談

```
> ちょっと壁打ちしたいんだけど、料金体系で悩んでて
秘書: いいですね！聞かせてください。
      今考えている選択肢はありますか？

> 月額制かBuyout（買い切り）か迷ってる
秘書: それぞれのメリットを整理してみましょう...
      ...
      結論をメモに保存しておきましょうか？
```

壁打ちの結果は `secretary/notes/` に保存されます。

## クイックメモ

```
> あ、さっき思いついたんだけど、LPにFAQセクション追加したい
秘書: メモしておきますね！
  → secretary/inbox/2026-03-16.md に記録
```

思いつきやアイデアは `secretary/inbox/` にキャプチャされます。

## 自動記録

秘書は以下を言われなくても自動で記録します：

| 内容 | 保存先 |
|------|--------|
| 意思決定 | `secretary/notes/YYYY-MM-DD-decisions.md` |
| 学び・気づき | `secretary/notes/YYYY-MM-DD-learnings.md` |
| アイデア | `secretary/inbox/YYYY-MM-DD.md` |
